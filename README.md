This is my personal blog about series, anime, videogames, etc...

# Tecnologies

- Notion
- i18next
- NextJS
- Typescript
- ChakraUI

# Start project

1. ``` $ git clone https://github.com/AnibalDBXD/mylink.git```
2. ``` $ cd mylink ```
4. ``` npm install ```
5. ``` npm run dev ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
